Function()treeNode.getCheckStatus

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

获取节点 checkbox / radio 半勾选状态。[setting.check.enable = true 时有效]

初始化节点数据时，由 zTree 增加此属性，请勿提前赋值

Function 参数说明

返回值JSON

{
	checked: true, //等同于 treeNode.checked
	half: true  //规则见下表
}
